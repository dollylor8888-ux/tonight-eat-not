# 「今晚食唔食」Web Wireframe（PWA）— MVP v1

> 目標：免費 Web 版做到 **1 秒回覆**；可以「加入主畫面」當 App 用；後續可無痛升級 WhatsApp Pro。

---

## 0. 全局設計原則（MVP）

* **單手操作**：主要按鈕置底、超大。
* **一屏完成**：Home 不要滾動（或最多 1 次小滾動）。
* **長輩友好**：字體大、對比高、少文字。
* **香港語氣**：用「返唔返嚟」「今晚」「幾多個」。

## 3) Onboarding（首次登入流程）

### 3.1 Create / Join 家庭（/onboarding/family）

兩個卡片選項：

**A 建立家庭**

* 家庭名稱（例：陳家 / Wong Family）

* 按鈕：建立

**B 加入家庭**

* 輸入邀請碼 / 點邀請連結自動加入

```
┌──────────────────────────────┐
│  建立你嘅家庭                  │

│                              │
│  家庭名稱： [ ________ ]        │

```

## 5) 今日頁（/app/today）— 核心

### 5.1 今日狀態卡（第一屏重點）

```
┌──────────────────────────────┐
│  今晚（2 月 21 日｜五）             │
│  目前：2 人 ✅ ｜1 人 ❌ ｜1 人 ⏰
└──────────────────────────────┘
```

### 5.2 成員列表（可一屏顯示）

每個成員 row：

* Avatar（首字母） + 名稱
* 右側狀態 pill：✅/❌/⏰

```
┌──────────────────────────────┐
│  A 媽媽            ✅ 會         │
│  B 爸爸            ⏰ 未知        │
│  C 阿仔            ❌ 唔會        │
│  D 阿女            ✅ 會         │
└──────────────────────────────┘
```

### 5.3 置底超大按鈕（你自己回覆）

固定在底部（sticky bar）：

* ✅ 會
* ❌ 唔會
* ⏰ 未知

按下後：

* 即時更新自己的 row
* 顯示 1 秒 Toast：「已更新：✅ 會」

### 5.4（可選）提醒未回覆（先灰掉，Pro 才解鎖）

* 「提醒未回覆」按鈕（顯示鎖 icon）
* 點擊彈出 Upsell（見第 9 節）
